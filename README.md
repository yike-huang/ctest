# Inner Canvas

Inner Canvas is a first-version static website prototype for an art-based emotional reflection app for people living with and beyond cancer.

## Open Locally

Open `index.html` directly in a browser.

## GitHub Pages

This site is GitHub Pages friendly because it uses plain static files:

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

In GitHub, enable Pages for the repository branch and root folder. The website should load without a build step.
